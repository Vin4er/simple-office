Вызов редактора:

##  $(jQuerySelector).se(); // все настройки - по дефолту
------------------------------------------------------------------------------------------

Редактор - принимает в себя объект, который имеет свойства, хранящие следующие настройки:

##  $(jQuerySelector).se({
##  
##  	  /* кнопки */
##        buttons: {
##            // Жирный, наклонный, подчеркнутый, по левому,    по центру,       по правму,   верхний регистр,
##  	      'default': ['bold','italic', 'underline','justifyleft', 'justifycenter', 'justifyright','superscript',
##                        //нижний регистр, нумерованный,     не нумерованный списки
##                           'subscript','InsertUnorderedList','InsertOrderedList'],
##  
##            /* Цвет текста, Цвет фона */ // Для использования типа colorpicker, должен быть подключен colorpicker.jquery.js и стили se-colorpicker.css
##  	      'colorpicker': ['ForeColor', 'BackColor'], 
##  	      
##            /*Шрифт, размер шрифта*/
##  	      'font-type': ["FontName", "FontSize"], 
##  
##  	      'table': ['insert-table'], // таблицы
##       // Каждый элементы можно удалить или перемешать(только внутри своей группы default, font-type, colorpicker, table)   
##       },
##  
##       /*нажатые кнопки на тулбаре*/
##       cssButton: { 
##  	     css: {	// нажатое состояние
##  		    "outline" : "1px solid rgba(0, 0, 0, .5)" 
##  	     }, 
##  	     def: { // отжатое состояние
##              "outline" : "none"
##           }	 
##       },
##  
##       /* Список шрифтов для редактора*/
##       fontsName: ['Arial', 'Tahoma', 'Impact', 'Georgia', 
##  				'Courier', 'Verdana', 'Courier New', 'Times New Roman', 'Arial Black'],
##  });

Вызов колрпикера (для использоавния в воих нуждах):
##  $(jQuerySelector).seColorpicker(); // все настройки - по дефолту. jQuerySelector - объъект при клике на который появляетя колорпикер
------------------------------------------------------------------------------------------

колрпикер - принимает в себя объект, который имеет свойства, хранящие следующие настройки:

##  $(jQuerySelector).seColorpicker({
##       selector: $("body"), // jQuery тег, у которого изменяем свойства
##       change: function(){  // метод для изменения цвета
##        	console.log(this.color);
##       },
##       close: function(){ // метод при зактрытии колорпикера
##          console.log("colorpicker-closet");
##       },
##       style_color: "background-color", // css свойство, которое надо изменять
##       color: "#ffffff", // цвет по умолчанию 
##  })
